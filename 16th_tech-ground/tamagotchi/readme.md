## Guide
### Claude CLI

Install claude-cli
```bash
curl -fsSL https://claude.ai/install.sh | bash
```

Execute claude-cli
```bash
echo 'export claude="$HOME/.local/bin/claude"' >> ~/.bashrc
source ~/.bashrc

echo 'export ANTHROPIC_API_KEY="{ANTHROPIC_API_KEY}"' >> ~/.bashrc
source ~/.bashrc

claude
```
